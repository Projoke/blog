# 如何使用Linux帮助文档
## help option
- 建议使用long option `df -h` `cp -h` `cd -h`
- -h 的含义 `man ls` `man df`

## help
-  局限性 `help is a shell builtin`
- 应用范围 `shell builtin`
- Ex: `help {` `help .`
## info
- GNU Software `info` `info info` `info coreutils`

## man
	了解它， 爱上它，征服它
-  文件是分类的  `man read`
-  man 文件目录
-  查看软件的man 文件
